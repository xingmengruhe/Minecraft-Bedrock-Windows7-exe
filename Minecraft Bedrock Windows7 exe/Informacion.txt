Preguntas Frecuentes:

- ¿Por qué el juego me crashea al iniciar? -
Esto pasa porque no tienes una actualizacion importante de windows 7 instalala y prueba
https://www.microsoft.com/es-ES/download/details.aspx?id=36805

- El juego me dice que falta una API ¿Qué hago? -
Esto pasa porque no tienes Visual C++ instalado. Busca en google el programa y instalalo
en concreto la version de 2015
https://www.mediafire.com/file/2bjxzf59duumsr5/Instalador_Visual_C%252B%252B_Net_Framework_y_DirectX.zip/file

- Cuando abro el juego me dice "Ha ocurrido un error" y no me carga el juego ¿Que hago? -

Esto pasa ya que el MceeLoginSkip (programa que usamos para crackear el juego) no funcionó correctamente
una de las soluciones es desactivar el antivirus o descargar el juego de nuevo

-¿Porque me detecta como virus el juego?

Porque necesitamos un programa llamado MceeLoginSkip/1-Minecraft para crackear el juego
este programa modifica la ram de minecraft, para crackearlo y jugar sin necesidad de comprar el juego
pero lamentablemente los modificadores de ram siempre se detectan como virus como cheat engine
también este programa es de código abierto y lamentablemente lo eliminaron de github 

-¿Porque no hay versiones nuevas?
Lamentablente este proyecto esta basado en minecraft education edition, y tenemos que esperar
a que lanzen una actualizacion a esta version. Esa es la razon
